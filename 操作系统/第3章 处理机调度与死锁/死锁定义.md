# 死锁定义

## 概念
**死锁（Deadlock）** 是指多个进程因**竞争资源**而造成的一种**僵局**，若无外力作用，这些进程都将**永远无法继续执行**。

## 死锁与饥饿的区别
| 概念 | 说明 |
|------|------|
| **死锁** | 进程互相等待对方释放资源，阻塞不前 |
| **饥饿** | 进程因优先级低等一直得不到所需资源 |

## 死锁示例
```
进程 A 持有资源 R1，等待资源 R2
进程 B 持有资源 R2，等待资源 R1
→ 两者都无法继续
```
