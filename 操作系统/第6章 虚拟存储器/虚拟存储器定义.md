# 虚拟存储器定义

## 概念
**虚拟存储器**是指具有**请求调入**和**置换**功能，能从逻辑上对内存容量加以扩展的存储器系统。

## 核心思想
- 程序不需要全部装入内存即可运行
- 只装入当前需要的部分
- 其余部分在需要时从外存调入

## 特征
| 特征 | 说明 |
|------|------|
| **多次性** | 作业分多次调入内存 |
| **对换性** | 进程运行时可在内存外存间对换 |
| **虚拟性** | 从逻辑上扩充内存容量 |

## 实现方式
- **请求分页**：基于分页的虚拟存储器
- **请求分段**：基于分段的虚拟存储器
