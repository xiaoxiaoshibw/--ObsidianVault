# 页面置换综合题

## 典型题型
给定页面引用序列和物理块数，计算缺页次数和缺页率。

## 示例
```
页面引用序列：7, 0, 1, 2, 0, 3, 0, 4, 2, 3, 0, 3, 2, 1, 2, 0, 1, 7, 0, 1
物理块数：3
```

### FIFO
| 步骤 | 7 | 0 | 1 | 2 | 0 | 3 | 0 | 4 | 2 | 3 | 0 | ... |
|------|---|---|---|---|---|---|---|---|---|---|---|-----|
| 块1 | 7 | 7 | 7 | 2 | 2 | 2 | 2 | 4 | 4 | 4 | 0 | ... |
| 块2 |   | 0 | 0 | 0 | 0 | 3 | 3 | 3 | 2 | 2 | 2 | ... |
| 块3 |   |   | 1 | 1 | 1 | 1 | 0 | 0 | 0 | 3 | 3 | ... |
| 缺页 | ✓ | ✓ | ✓ | ✓ |   | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ... |

### LRU
| 步骤 | 7 | 0 | 1 | 2 | 0 | 3 | 0 | 4 | 2 | 3 | 0 | ... |
|------|---|---|---|---|---|---|---|---|---|---|---|-----|
| 块1 | 7 | 7 | 7 | 2 | 2 | 2 | 2 | 4 | 4 | 4 | 0 | ... |
| 块2 |   | 0 | 0 | 0 | 0 | 0 | 0 | 0 | 2 | 2 | 2 | ... |
| 块3 |   |   | 1 | 1 | 1 | 3 | 3 | 3 | 3 | 3 | 3 | ... |
| 缺页 | ✓ | ✓ | ✓ | ✓ |   | ✓ |   | ✓ | ✓ |   | ✓ | ... |

> 做题要点：分清**缺页**和**置换**——只有内存块满且页面不在内存时才需要置换。
> 缺页次数 = 首次调入 + 置换次数
