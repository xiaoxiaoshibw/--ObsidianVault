# I/O 控制方式

## 四种方式

| 方式 | CPU 参与度 | 数据传输单位 | 特点 |
|------|-----------|-------------|------|
| **程序查询** | 全程 | 字/字节 | CPU 主动循环检查，效率最低 |
| **中断** | 发起+结束 | 字/字节 | CPU 不用等待，但中断频繁 |
| **DMA** | 仅初始化 | 块 | 硬件直接控制内存↔设备传输 |
| **通道** | 极少 | 块/组 | 专用处理器执行通道程序 |

## 发展趋势
```
程序查询 → 中断 → DMA → 通道
                   ↓
          CPU 干预越来越少
          数据传输效率越来越高
```

## 📝 本章习题
**选择题2**：**键盘**属于字符设备。（选 D）
- A. 磁盘 ❌
- B. 磁带 ❌
- C. 光盘 ❌
- D. 键盘 ✅

**选择题3**：**磁盘**属于块设备。（选 B）
- A. 键盘 ❌
- B. 磁盘 ✅
- C. 显示器 ❌
- D. 打印机 ❌

**填空题3**：从设备分配的观点看，可将设备分为**独占**设备和**共享**设备和虚拟设备三类。

**填空题4**：操作系统的设备管理应具备的主要功能：为用户提供和设备管理相关的**接口**、进行**设备分配**、实现设备和设备以及设备和CPU等之间**并行操作**、进行**缓冲区**管理等。

**填空题6**：从资源分配的角度看，可以把设备分为独占设备和共享设备。打印机属于**独占**设备，而磁盘属于**共享**设备。

**填空题8**：在UNIX系统中，所有的**设备文件**都被看成是特殊文件，它们在使用形式上与普通文件相同，但它们使用是和**设备驱动程序**紧密相连的。

**填空题10**：独占设备是指在一个作业的整个执行期间独自占用的设备，它一般采用**静态**分配。共享设备是指在某个时间段内可由多个作业同时使用的设备，一般采用**动态**分配。

**填空题13**：磁盘属于**块**设备，其信息的存取是以**数据块**为单位的；磁盘的I/O控制主要采取**DMA**方式；打印机的I/O控制主要采取**中断**方式。
