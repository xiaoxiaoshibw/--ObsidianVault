# SCAN 电梯算法

## 定义
**SCAN 算法**也称为**电梯算法**：磁头沿一个方向移动，处理沿途的所有请求，到达终点后**掉头**。

## 特点
| 项目 | 内容 |
|------|------|
| 策略 | 单向移动，处理沿途请求 |
| 优点 | 不会饥饿，磁头移动距离较合理 |
| 缺点 | 到达终点才掉头，不够灵活 |

## 工作过程
```
磁头从 0 向 199 方向移动（假设）
沿途处理所有请求
到达 199（或最远请求）
掉头向 0 方向移动
沿途处理所有请求
```

## 对比 SSTF
- SCAN 避免了 SSTF 的**饥饿**问题
- 但 SCAN 到边缘才掉头，可能浪费
