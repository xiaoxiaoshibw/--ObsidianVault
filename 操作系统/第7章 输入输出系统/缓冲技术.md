# 缓冲技术

## 引入目的
1. 缓解 **CPU 与 I/O 设备速度不匹配**的矛盾
2. 减少对 CPU 的中断频率
3. 解决数据粒度不匹配问题

## 缓冲类型
| 类型 | 说明 |
|------|------|
| **单缓冲** | 一个缓冲区，CPU 和设备交替使用 |
| **双缓冲** | 两个缓冲区，可同时进行输入和处理 |
| **循环缓冲** | 多个缓冲区组成的环形结构 |
| **缓冲池** | 系统维护的多个缓冲区集合 |

## 📝 本章习题
**填空题1**（第5章）：操作系统中采用缓存技术的主要目的是提高CPU和设备之间的**并行操作**程度。

**填空题9**（第6-7章）：在利用RS-232接口进行通信时，其通信速率为9.6KB/S(B为Bit)。如果在通信接口中仅设置了一个8位寄存器作为缓冲寄存器，这意味着大约每隔 **0.833ms** 的时间便要中断一次CPU，且要求CPU必须在 **0.833ms** 时间内予以响应。
> 解：9.6Kb/s = 9600b/s，1字节=8位，传输1字节时间 = 8/9600 ≈ 0.833ms
