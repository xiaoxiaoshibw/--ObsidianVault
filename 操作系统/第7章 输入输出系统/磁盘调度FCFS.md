# 磁盘调度 - FCFS

## 定义
**先来先服务**：按照磁盘请求的**到达顺序**依次处理。

## 特点
| 项目 | 内容 |
|------|------|
| 调度策略 | 按请求顺序 |
| 优点 | 简单、公平 |
| 缺点 | 性能差，磁头移动距离大 |
| 适用 | 请求较少时 |

---

## 📝 本章习题
**填空题1**（第6-7章）：影响磁盘读写时间的因素有**寻道时间**、**旋转延迟时间**、**传输时间**。

**填空题11**（第6-7章）：转速为7200转/分钟，平均旋转延迟时间约为 **4.17ms**。
> 解：旋转一周时间 = 60/7200 ≈ 8.33ms，平均旋转延迟 = 8.33/2 ≈ 4.17ms
