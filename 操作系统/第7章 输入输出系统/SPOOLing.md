# SPOOLing 技术

## 定义
**SPOOLing（Simultaneous Peripherals Operation On-Line）** 也称为**假脱机技术**，是多道程序设计中的一种**虚拟设备**技术。

## 核心思想
将**独占设备**改造成**共享设备**。

## 组成
| 组件 | 说明 |
|------|------|
| **输入/输出井** | 磁盘上开辟的存储区域 |
| **输入/输出缓冲区** | 内存中的缓冲区 |
| **预输入程序** | 将数据从设备提前读入磁盘 |
| **缓输出程序** | 将磁盘中的数据输出到设备 |

## 典型应用
- **共享打印机**：多个用户同时提交打印任务
- 将物理的独占打印机 → 虚拟的共享打印机

## 好处
- 实现了**虚拟设备**功能
- 提高了 I/O 速度
- 将独占设备变为共享设备
