# DMA 方式

## 定义
**DMA（Direct Memory Access）** 方式允许外部设备和内存之间**直接传输数据**，无需 CPU 逐字干预。

## 工作流程
```
CPU 初始化 DMA 控制器
  └─ 设置：传输方向、内存地址、传输字节数
  ↓
DMA 控制器接管总线，直接传输数据
  ↓
传输完成 → DMA 发送中断信号
  ↓
CPU 处理中断
```

## 特点
| 对比 | 中断方式 | DMA 方式 |
|------|----------|----------|
| 传输单位 | 字/字节 | **块** |
| 中断频率 | 每次传输都中断 | 整个块传输完才中断 |
| CPU 干预 | 每次传输都需要 | 仅初始化+结束后处理 |
| 数据传输 | CPU 负责 | **DMA 控制器负责** |
