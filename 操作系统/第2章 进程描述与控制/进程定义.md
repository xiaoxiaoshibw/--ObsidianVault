# 进程定义

## 概念
**进程（Process）** 是程序在**并发环境**中的一次**执行过程**，是系统进行**资源分配和调度**的**基本单位**。

## 进程 vs 程序

| | 进程 | 程序 |
|--|------|------|
| **动态性** | 动态（执行态） | 静态（磁盘上） |
| **生命周期** | 有创建→运行→消亡 | 长期存在 |
| **组成** | 程序+数据+PCB | 指令序列 |
| **一对一？** | 一个程序可对应多个进程 | 一个程序可产生多个进程 |

## 进程的本质
- **动态**：由创建而产生，由调度而执行，由撤销而消亡
- **并发**：多个进程交替占用 CPU
- **独立**：资源分配的基本单位
- **异步**：独立、不可预知的速度向前推进

## 📝 本章习题
**选择题**（进程映像）：进程的映像是 **PCB结构+程序+数据集**。
- A. 由协处理器执行的一个程序 ❌
- B. 一个独立的程序+数据集 ❌
- C. PCB结构+程序+数据集 ✅
- D. 一个独立的程序 ❌

**填空题2**：进程的执行并不是"一气呵成"，而是走走停停的，这种特征称为进程的**异步性**。

**填空题3**：进程的并发性是指多个进程在**同一时间间隔**内同时发生。
