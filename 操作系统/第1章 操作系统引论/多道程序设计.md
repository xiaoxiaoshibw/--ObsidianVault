# 多道程序设计

## 概念
允许多个程序同时进入内存并交替执行，**提高 CPU 利用率**。

## 特点
1. **多道**：内存中同时存放多道程序
2. **宏观上并行**：多道程序同时在运行
3. **微观上串行**：单 CPU 下交替执行

## 优点
- 提高 CPU 利用率
- 提高系统吞吐量

## 需解决的问题
- 资源分配问题
- 进程同步问题
- 内存保护问题
