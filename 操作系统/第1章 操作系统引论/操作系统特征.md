# 操作系统特征

## 四大特征

| 特征 | 说明 |
|------|------|
| **并发（Concurrency）** | 多个程序在同一时间间隔内交替执行 |
| **共享（Sharing）** | 资源可被多个进程共同使用（互斥/同时） |
| **虚拟（Virtual）** | 将物理实体映射为多个逻辑对应物（虚拟内存、虚拟CPU） |
| **异步（Asynchronism）** | 进程的执行顺序和执行时间不可预知 |

> 并发和共享是操作系统的**最基本特征**，两者互为存在条件。
