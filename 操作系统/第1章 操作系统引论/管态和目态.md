# 管态和目态

## 管态（系统态、内核态）
- 可以执行**所有指令**（包括特权指令）
- 可以访问所有资源
- 对应 CPU 的**核心态**

## 目态（用户态）
- 只能执行**非特权指令**
- 不能直接访问硬件资源
- 对应 CPU 的**用户态**

## 状态转换
- 目态 → 管态：**中断/异常/系统调用**
- 管态 → 目态：**修改 PSW**（特权指令）

- C. 核心态 ✅
- D. 就绪态 ❌

**判断题1**：处理机从管态转变为目态是通过置程序状态字来实现的。 → **正确** ✅

**判断题5**：处理机从目态转变为管态是通过置程序状态字来实现的。 → **错误** ❌
> 目态→管态通过中断/异常/系统调用实现，不是通过置PSW。

**简答题（系统模式）**：管态是操作系统内核运行的状态，可执行全部指令，包括特权指令；目态是用户程序运行的状态，只能执行非特权指令，不能直接使用系统资源。二态由程序状态字 PSW 中的状态位来区分。
