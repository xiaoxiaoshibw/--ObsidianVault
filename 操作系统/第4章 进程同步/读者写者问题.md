# 读者-写者问题

## 问题描述
- 多个进程共享一个数据文件
- **读者（Reader）**：只读，可同时多个
- **写者（Writer）**：写操作，必须独占

## 约束规则
1. **读-读**：允许（多个读者可同时读）
2. **读-写**：互斥（读者写者不可同时）
3. **写-写**：互斥（一写时不能有另一写）

## 伪代码实现（读者优先）

```
semaphore rw_mutex = 1;   // 读写互斥
semaphore mutex = 1;      // 保护 read_count
int read_count = 0;       // 读者数量

// 写者
writer() {
  P(rw_mutex);            // 请求写
    写入文件;
  V(rw_mutex);            // 释放写
}

// 读者
reader() {
  P(mutex);
    read_count++;
    if (read_count == 1)
      P(rw_mutex);        // 第一个读者阻止写者
  V(mutex);
    读取文件;
  P(mutex);
    read_count--;
    if (read_count == 0)
      V(rw_mutex);        // 最后一个读者释放写者
  V(mutex);
}
```

> **读者优先**：只要有读者正在读，后续读者可直接进入，写者可能饥饿。
