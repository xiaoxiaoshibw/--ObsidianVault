# PV 操作

## 定义
**P、V 操作**是信号量的两个**原子操作**（原语）。

## 操作含义

### P 操作（Wait / down / 申请资源）
```
P(S):
  S.value = S.value - 1
  if (S.value < 0) then
    进程进入等待队列
```

### V 操作（Signal / up / 释放资源）
```
V(S):
  S.value = S.value + 1
  if (S.value <= 0) then
    唤醒等待队列中的第一个进程
```

## 记忆方法
- **P** = Proberen（荷兰语：尝试）= 申请 → 减 1
- **V** = Verhogen（荷兰语：增加）= 释放 → 加 1

## 互斥模式
```
P(mutex)    // 进入区
  临界区
V(mutex)    // 退出区
```
