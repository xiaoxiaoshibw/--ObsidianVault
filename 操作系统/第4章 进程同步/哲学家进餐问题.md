# 哲学家进餐问题

## 问题描述
- 5 个哲学家围坐圆桌
- 每两人之间放一根筷子（共 5 根）
- 哲学家：思考 → 饿了 → 拿筷子 → 吃饭 → 放筷子 → 思考

## 可能的问题
**死锁**：每个哲学家都拿起左边筷子，等待右边筷子 → 全部阻塞

## 解决方案

### 方法1：最多允许4人同时吃饭
```
semaphore eat_limit = 4;  // 限制同时进餐人数
```

### 方法2：奇数哲学家先拿左筷，偶数先拿右筷
```
if (i % 2 == 0) {
  P(chopstick[i]);         // 先左
  P(chopstick[(i+1)%5]);   // 后右
} else {
  P(chopstick[(i+1)%5]);   // 先右
  P(chopstick[i]);         // 后左
}
```

### 方法3：一次拿两根筷子（原子操作）
```
P(mutex);                   // 互斥拿筷子
  P(chopstick[i]);
  P(chopstick[(i+1)%5]);
V(mutex);
```
