# 生产者-消费者问题

## 问题描述
- **生产者**：生产数据放入缓冲区
- **消费者**：从缓冲区取出数据消费
- 缓冲区有大小限制

## 同步关系
1. 缓冲区**满**时，生产者等待（同步）
2. 缓冲区**空**时，消费者等待（同步）
3. 多个生产者/消费者互斥访问缓冲区（互斥）

## 伪代码实现

```
semaphore mutex = 1;    // 互斥访问缓冲区
semaphore empty = n;    // 空缓冲区数量
semaphore full = 0;     // 满缓冲区数量

// 生产者
producer() {
  while (true) {
    produce an item;
    P(empty);           // 申请空缓冲区
    P(mutex);           // 进入临界区
      将物品放入缓冲区;
    V(mutex);           // 退出临界区
    V(full);            // 增加满缓冲区
  }
}

// 消费者
consumer() {
  while (true) {
    P(full);            // 申请满缓冲区
    P(mutex);           // 进入临界区
      从缓冲区取物品;
    V(mutex);           // 退出临界区
    V(empty);           // 增加空缓冲区
    consume the item;
  }
}
```

> ⚠ **注意 P 操作的顺序**：先 P(资源信号量)，后 P(互斥信号量)，否则可能死锁！

## 📝 本章习题
**填空题7（PV操作大题）**：三个并发进程P1、P2和P3，两个单缓冲B1和B2，容量均为M。

信号量设置：
- empty1 = **m**, empty2 = **m** （空缓冲区数量）
- full1 = **0**, full2 = **0** （满缓冲区数量）
- mutex1 = **1**, mutex2 = **1** （互斥访问）

P1（输入→判断正负→B1或B2）：
```
if (data > 0) {
  P(empty2); P(mutex2);
  放数据data到B2中；
  V(mutex2); V(full2);
} else {
  P(empty1); P(mutex1);
  放数据data到B1中；
  V(mutex1); V(full1);
}
```

P2（取B1→加工→送B2）：
```
P(full1); P(mutex1);
从B1中取出数据data，然后处理成DATA；
V(mutex1); V(empty1);
P(empty2); P(mutex2);
将数据DATA放入B2中；
V(mutex2); V(full2);
```

P3（取B2→输出）：
```
P(full2); P(mutex2);
从B2中取出数据data；
V(mutex2); V(empty2);
打印数据data；
```
