# 页表

## 定义
**页表（Page Table）** 是操作系统为每个进程维护的一种数据结构，用于实现从**逻辑页号**到**物理块号**的映射。

## 页表项
| 字段 | 说明 |
|------|------|
| **页号** | 逻辑页编号（通常隐含）|
| **块号** | 物理块编号 |
| **状态位** | 该页是否在内存中 |
| **访问位** | 页面是否被访问过 |
| **修改位** | 页面是否被修改过 |

## 地址转换过程
```
逻辑地址 → 提取页号 P → 查页表 → 得到块号 b
→ 物理地址 = b × 页大小 + 页内偏移
```
