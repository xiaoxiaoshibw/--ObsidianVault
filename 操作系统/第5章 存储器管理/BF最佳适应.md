# BF 最佳适应算法

## 定义
**Best Fit**：将空闲分区按**大小递增**排列，选择**最小的能满足要求**的空闲分区。

## 特点
| 项目 | 内容 |
|------|------|
| 排列方式 | 大小递增 |
| 策略 | 找最小可满足分区 |
| 优点 | 保留大空闲分区 |
| 缺点 | 产生大量小碎片（很难再利用）|

> 口诀：**BF 大小增**
