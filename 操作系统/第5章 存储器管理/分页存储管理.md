# 分页存储管理

## 基本概念
| 概念 | 说明 |
|------|------|
| **页（Page）** | 逻辑空间划分的单位 |
| **块/页框（Frame）** | 物理空间划分的单位 |
| **页大小 = 块大小** | 一一对应 |

## 特点
- **消除外部碎片**
- 存在**内部碎片**（最后一页可能不满）

## 地址结构
```
┌──────────────┬────────────┐
│   页号 (P)   │ 页内偏移 (W) │
└──────────────┴────────────┘
```

## 地址转换
```
逻辑地址 → 查页表 → 物理块号 + 页内偏移 → 物理地址
```

> **主存分配单位**：**块（页框）** ✅

## 📝 本章习题
**选择题3**：在页式存储管理方案中，进行主存分配的单位是**块**。（选 B）
- A. 段 ❌
- B. 块 ✅
- C. 作业 ❌
- D. 不一定 ❌

**填空题3**：在具有两级页表的分页存储管理系统中，CPU每次要存取一个数据时，必须访问 **3** 次内存。

**填空题9**：已知一个进程具有页号0、1、2，分别对应内存块号为2、3、6，页面大小为1K，其中某一指令对虚地址为1000（十进制地址）进行存取操作，则对应的物理内存地址为 **3048**。
> 解：1000 ÷ 1024 = 0 余 1000 → 页号0 → 块号2 → 物理地址 = 2×1024 + 1000 = 3048

**填空题12**：系统中有4MB内存，最大有效物理地址为 **3FFFFFH**，如果采用分页管理，页长1KB，全部内存可分为 **4096** 个页帧（块）。
> 解：4MB = 4×2²⁰ = 2²²，最大地址 = 2²²-1 = 3FFFFFH；4MB÷1KB = 4096

**填空题13**：在一个分页存储管理的系统中，页长为4KB，某一作业的页表中，虚拟地址3000对应物理地址16进制值为 **1BB8H**，12000对应物理地址16进制值为 **3EE0H**。
> 解：页号=3000÷4096=0 → 块号1 → 物理=1×4096+(3000-0)=7096=1BB8H
> 12000÷4096=2 → 块号6(根据页表) → 物理=6×4096+(12000-2×4096)=16096=3EE0H

**填空题15**：分页系统的页大小为1KB，16进制的虚拟地址0x3C8F，对应的页号为10进制值 **15**，页内地址为16进制值 **08FH**。
> 解：0x3C8F = 15499，15499÷1024=15余139，139=08FH
