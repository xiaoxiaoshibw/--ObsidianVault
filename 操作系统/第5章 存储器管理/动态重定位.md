# 动态重定位

## 定义
**动态重定位**是在**执行过程中**进行的地址转换，依靠**重定位寄存器**实现。

## 特点
- 地址转换在运行时逐条指令进行
- 依靠**重定位寄存器（基址寄存器）**
- 作业可在内存中移动（紧凑）

## 流程
```
逻辑地址 + 重定位寄存器的值 = 物理地址
```

## 优缺点
| 优点 | 缺点 |
|------|------|
| 支持不连续存储 | 需要硬件支持 |
| 支持程序浮动（紧凑） | 实现稍复杂 |

> ✅ **高频考点**：动态重定位发生于**作业执行过程中**。
