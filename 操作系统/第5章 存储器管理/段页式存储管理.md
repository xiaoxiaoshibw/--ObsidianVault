# 段页式存储管理

## 概念
**段页式** = **分段** + **分页**的混合管理方式。

## 结构
```
逻辑地址：段号 + 段内页号 + 页内偏移
```

## 地址转换
```
段号 → 查段表 → 得到页表起始地址
页号 → 查页表 → 得到物理块号
物理地址 = 块号 × 页大小 + 页内偏移
```

## 特点
| 优点 | 缺点 |
|------|------|
| 兼具分段的逻辑性和分页的物理性 | 地址转换需要查两次表（开销大）|
| 无外部碎片 | 段表+页表占用额外空间 |
| 便于共享和保护 | 硬件支持复杂 |

## 📝 本章习题
**选择题4**：在段页式存储管理系统中，当访问主存中的一条指令或数据时**至少访问三次主存**。（选 C）
- A. 需访问两次主存 ❌
- B. 需访问一次主存 ❌
- C. 至少访问三次主存 ✅
- D. 至少访问两次主存 ❌
> 解：段表 → 页表 → 数据，共3次

**填空题10/11**：段页式存储管理中访问主存中的一条指令或存取数据，至少需访问内存 **3** 次。内存管理的三种基本方式有**分区存储管理**、**分页存储管理**、**分段存储管理**。
