# 快表 TLB

## 定义
**TLB（Translation Lookaside Buffer）** 也称为**快表**，是一种**高速缓存**，用于加速地址转换。

## 原理
- 存放最近使用过的页表项
- 基于**局部性原理**
- 先查 TLB（硬件），命中则快速转换
- TLB 未命中时查内存中的页表

## 带 TLB 的地址转换流程
```
1. 在 TLB 中查找页号
2. 若命中（TLB 命中）→ 直接得到块号
3. 若未命中 → 查内存页表，同时更新 TLB
```

## 性能
- TLB 命中率 > 99% 时，性能接近纯硬件地址转换
- 上下文切换时需刷新 TLB

- B. 保存最近用过的页表项 ✅
- C. 保存最近用过的物理地址 ❌
- D. 保存最近用过的虚拟地址 ❌

**填空题4**：分页存储管理系统具有快表，内存访问时间为2μs，检索快表时间为0.5μs。若快表的命中率为80%，且忽略快表更新时间，则有效访问时间是 **2.9** μs。
> 解：0.5 + 0.8×2 + 0.2×(2+2) = 0.5 + 1.6 + 0.8 = 2.9μs
> （先查TLB 0.5μs，命中则再2μs访问内存；未命中则再2μs查页表+2μs访问数据）
