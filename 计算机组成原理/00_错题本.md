# ⚠️ 易错题本

## 计算机系统概述
1. **CPI与时钟频率无关**（与系统结构、指令集、计算机组织有关）
2. 机器字长 ≠ 指令字长 ≠ 存储字长（**可能不等**）
3. 计算机字长**不一定**都是32位（16、32、64位都可能）
4. 解释程序速度**慢**（不是速度快！）
5. MFLOPS用于科学计算（不是MIPS）

## 数据表示
1. **0的表示唯一**的是补码和移码（原码和反码都有两个0）
2. 8位原码表示255个不同数（不是256），补码才是256
3. -66的补码是**BEH**（不是C2H）
4. 主存地址用**无符号数**表示
5. 1.2**不能**用IEEE754精确表示（二进制无限循环）

## 存储系统
1. DRAM地址线**减半**（地址复用），SRAM不减半
2. Cache存储系统总容量 = **主存容量**（不是两者相加）
3. Cache功能全部由**硬件**实现
4. Cache替换单位是**块**（不是字）
5. Cache与主存交换数据单位是**块**，CPU与Cache交换是**字**
6. 页表不命中 → Cache一定也不命中

## 指令系统
1. **间接寻址**不属于偏移寻址方式
2. 变址寻址适合数组访问
3. 相对寻址EA = (PC) + 偏移量

## 中央处理器
1. CPU不包含**地址译码器**
2. **IR**对程序员透明（不可见），**PC**可见
3. 取指令**不受操作码控制**
4. 硬布线速度**快**，微程序速度**慢**
5. 控制存储器（CS）在**CPU内**，用**ROM**实现
6. 微程序控制器时序系统**简单**（不是复杂）
