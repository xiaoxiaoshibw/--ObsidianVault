# 数制转换

## 常考例题
- 十进制-66的8位补码 → **BEH**
  66 = 0100 0010 → 取反 1011 1101 → +1 → 1011 1110 = BEH

- 9BH（补码）→ 十进制
  9BH = 1001 1011 → 取反 1110 0100 → +1 → 1110 0101 = -101

## 定点数
- 最高位为符号位（0正1负）
- n+1位定点整数原码范围：hBc(2^n-1) \leq x \leq 2^n-1$
- n+1位补码最大值：^n-1$（符号位0，数值位全1）

## IEEE754单精度浮点数
 = (-1)^S 	imes 1.M 	imes 2^{E-127}$
- S: 1位, E: 8位（移码）, M: 23位
- **不能精确表示**：1.2（循环小数）
