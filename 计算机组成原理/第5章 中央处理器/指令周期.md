# 指令周期

## 基本概念
| 周期 | 说明 |
|------|------|
| **时钟周期** | 最小时间单位 = 1/主频 |
| **机器周期** | = **存取周期**（访问一次主存的时间） |
| **指令周期** | 取指令 + 分析 + 执行 |

## 顺序关系
指令周期 ≥ 机器周期 ≥ 时钟周期

## 取指周期
- 第一个机器周期总是**取指周期**
- 取指令是控制器固有功能，**不受操作码控制**

## 间址周期
间址周期结束后，MDR中存放的是**操作数地址**

## 区分指令和数据
**指令周期的不同阶段**
- 取指阶段：取的是指令
- 执行阶段：取的是数据
