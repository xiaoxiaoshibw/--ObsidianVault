# CPU 组成

## CPU 两大部件
**运算器** + **控制器**

## 运算器
- **ALU**（算术逻辑单元）— 组合逻辑电路
- 通用寄存器组
- 状态寄存器（PSW）

## 控制器
- **PC**（程序计数器）— 存下条指令地址
- **IR**（指令寄存器）— 存当前执行指令
- 时序电路
- 不包括：**程序状态字寄存器**

## 程序员可见的寄存器
- **PC**（可通过转移指令设置）
- 通用寄存器
- 状态寄存器（标志寄存器）
- **IR**、**MAR**、**MDR** 对程序员**不可见**（透明）

## n位CPU的含义
**数据总线线数** = n

## CPU不包含的部件
**地址译码器**（属于主存）
