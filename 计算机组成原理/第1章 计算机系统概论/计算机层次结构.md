# 计算机层次结构

## 硬件与软件
- **完整的计算机系统** = 配套的**硬件设备**和**软件系统**
- 主机 = CPU + 主存
- 运算器包含：状态寄存器、数据总线、ALU（不包含**地址寄存器**）

## 程序设计语言层次

| 语言 | 特点 |
|------|------|
| **机器语言** | 计算机唯一可直接执行的语言 |
| **汇编语言** | 用助记符编写，与机器结构**相关**（不可移植） |
| **高级语言** | 需编译/解释后执行 |

## 编译程序 vs 解释程序

| 特性 | 编译程序 | 解释程序 |
|------|---------|---------|
| 翻译方式 | 一次翻译全部 | 逐句翻译执行 |
| 目标文件 | 生成目标代码文件 | 不生成 |
| 速度 | 运行速度**快** | 速度**慢** |
| 代表 | C, C++ | JavaScript, Python |

- 高级语言→可执行文件：**预处理→编译→汇编→链接**
