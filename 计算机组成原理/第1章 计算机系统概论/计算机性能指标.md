# 计算机性能指标

## 常用指标

| 指标 | 含义 | 公式 |
|------|------|------|
| **CPI** | 执行一条指令所需时钟周期数 |  = 时钟周期数/指令条数$ |
| **MIPS** | 每秒百万条指令 |  = 主频(MHz)/CPI$ |
| **MFLOPS** | 每秒百万次浮点运算 | 科学计算最有用参数 |
| **主频** | 时钟频率 | $时钟周期 = 1/主频$ |
| **执行时间** | 程序运行时间 |  = 指令数 	imes CPI / 主频$ |

## CPI与什么无关
**时钟频率**（与系统结构、指令集、计算机组织有关）

## 例题
**题1**：主频1.2GHz，各类指令比例及CPI：
- A(50%, CPI=2), B(20%, CPI=3), C(10%, CPI=4), D(20%, CPI=5)
- 平均CPI = 2×0.5 + 3×0.2 + 4×0.1 + 5×0.2 = **3**
- MIPS = 1200/3 = **400**

**题2**：程序P执行时间20s，优化后指令数减少到70%，CPI增加到1.2倍
- 新时间 = 20 × 0.7 × 1.2 = **16.8s**

**题3**：CPU时间90s，I/O时间10s，CPU速度提高50%
- 新时间 = 90/1.5 + 10 = **70s**

**题4**：M1(1.5GHz, CPI=2), M2(1.2GHz, CPI=1)，运行时间比
- M1时间 : M2时间 = (2/1.5) : (1/1.2) = **1.6**

## 机器字长
- **机器字长**：CPU运算器位数 = 通用寄存器宽度
- 指令字长：指令的二进制位数
- 存储字长：一个存储单元的位数 = **MDR的位数**
- 三者数值上**可能相等也可能不等**
- 计算机中一个字长可以是16、32、64位等，**不一定都是32位**
