# 存储器层次结构

## 层次结构
- **CPU寄存器** → **Cache** → **主存** → **辅存（磁盘）**
- CPU按**主存物理地址**访问
- Cache中存放的是主存中某部分信息的**副本**
- 存储系统总容量 = **主存容量**（不是简单的相加）

## 分类
| 类型 | 特点 | 举例 |
|------|------|------|
| RAM（随机存取） | 可读可写 | SRAM、DRAM |
| ROM（只读） | 断电不丢失 | Mask ROM、闪存 |
| SAM（顺序存取） | 按顺序访问 | 磁带 |
| DAM（直接存取） | 直接定位+顺序访问 | **磁盘** |

## 相联存储器
按**内容**或**地址**进行寻址（TLB、相联Cache）

## 存取周期
- **存取时间**：一次读或写的平均时间
- **存取周期**：连续两次访问的**最短时间间隔**
- 存取周期 = 存取时间 + 恢复时间
