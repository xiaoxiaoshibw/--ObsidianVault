# Cache 计算题

## 题型1：平均存取时间
Cache 2ns，主存 40ns，平均 3ns，求命中率
- $2h + 42(1-h) = 3$
- $h = 97.5\%$

## 题型2：Cache行位数
32位地址，32KB Cache，32B块，**直接映射**，回写
- 块内偏移：log₂(32) = 5位
- 行数：32KB/32B = 1024（行） → Index 10位
- Tag：32-5-10 = **17位**
- 行位数 = 32×8（数据）+ 17（Tag）+ 1（脏）+ 1（有效）= **275位**

## 题型3：四路组相联
- 组数 = 行数 / 路数
- Index位数 = log₂(组数)
