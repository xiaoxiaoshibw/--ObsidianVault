# 寻址方式

## 各类寻址方式

| 方式 | 有效地址 | 说明 |
|------|---------|------|
| **立即寻址** | 操作数在指令中 | 最快 |
| **直接寻址** | EA = A | 地址码即操作数地址 |
| **间接寻址** | EA = (A) | 需访存读地址 |
| **寄存器寻址** | 操作数在寄存器 | 最快 |
| **寄存器间接** | EA = (R) | 寄存器内容为地址 |
| **基址寻址** | EA = (BR) + A | 支持程序重定位 |
| **变址寻址** | EA = (IX) + A | 适合数组访问 |
| **相对寻址** | EA = (PC) + A | 转移指令 |

## 偏移寻址方式
包括：**基址寻址、相对寻址、变址寻址**
不包括：**间接寻址**

## 变址寻址
最适合**按下标顺序访问一维数组元素**

## 先变址后间址
EA = ((I) + D)
